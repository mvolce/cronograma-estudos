# Cronograma de Estudos

Um projeto front-end simples para organizar seus estudos de forma prática.  
Permite adicionar atividades e marcar quando elas forem concluídas.

---

##  Funcionalidades

- Adicionar atividades ou matérias no cronograma  
- Marcar atividades como concluídas  

> Observação: Projeto apenas front-end, roda diretamente no navegador.

---

##  Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript
